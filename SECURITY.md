# Security Policy

## Supported Versions

This is currently updated, adhoc, into the `main` branch of the [frontend repository](https://github.com/wikisim/wikisim-frontend), [supabase repository](https://github.com/wikisim/wikisim-supabase) and [server repository](https://github.com/wikisim/wikisim-server) which provide the functionality for wikisim.org.
It is also used by other projects (e.g. visualisations, simulations, and interactables) hosted on wikisim and elsewhere.
Please inform us of any vulnerabilities in this repository by using the private "[Report a vulnerability](https://github.com/wikisim/wikisim-core/security/advisories/new)".

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |

## Reporting a Vulnerability

To report this vulnerability please use the private "[Report a vulnerability](https://github.com/wikisim/wikisim-core/security/advisories/new)"

Thank you for your message; there are no guaranteed response times as the team is currently one unpaid person.
If you don't receive a response in 5 days please repost.
If you think it is urgent the please repost up to every 12 hours or email us at wikisim dot secure at gmail dot com

If the vulnerability is accepted you'll get gratitude and, if you wish, a personal public thank you in the list on contributors to the code base.
